## 🧩 O PRINCÍPIO LEGO

Toda tool/habilidade nova é montada como LEGO:
- Pegar **VFX** de uma tool existente
- Pegar **SFX** de outra
- Pegar **funções** de uma terceira
- Pegar **mecânicas** de uma quarta
- Combinar tudo → criar algo único

**Sempre citar a origem de cada peça no comentário do script.**


## ⚔️ SISTEMA DE HABILIDADES PASSIVAS

### Regra crítica
> Aparecem **APENAS** quando o jogador selecionou um personagem.
> Sem personagem equipado = **zero botões na tela**.

### Fluxo de ativação
```
Jogador seleciona personagem
       ↓
CharacterSelectionServer: MorphCharacter()
       ↓
Insere BoolValue "CharacterSelected" no NewChar
       ↓
FireClient(player, "CharacterLocked")
       ↓
PassiveCombatClient detecta → ativa os 3 botões
```

### Scripts envolvidos
- `PassiveCombatServer` (ServerScriptService) — dano, knockback, bloqueio server-side
- `PassiveCombatClient` (StarterCharacterScripts) — botões, animações CFrame, VFX local
- `ReplicatedStorage/PassiveCombat/` — RemoteEvents: Punch, Block, Dash

### As 3 habilidades

**👊 Soco**
- Input: `E` (PC) | `ButtonX` (Console) | Botão tela (Mobile)
- Base: `Fists (Soco)` — SFX punch1-4, hitflesh1-3, BodyVelocity knockback
- Dano: 15 | Range: 6 | Cooldown: 0.5s
- Animação: CFrame lerp no Right Arm (extend → retract)

**🛡 Defesa (SEGURAR)**
- Input: `F` (PC, segurar) | `ButtonR2` (Console) | Botão tela (Mobile)
- Base: `escudo noob` — C0 welds exatos Rufus14, WalkSpeed × 0.7
- Reduz 80% do dano | 20% de chance de negar 100%
- Animação: welds rightarmw/headw/rootpartw/leftarmw com lerp RunService.Stepped

**💨 Dash**
- Input: `Q` (PC) | `ButtonCircle` (Console) | Botão tela (Mobile)
- Base: `dodge/dash (SKILL PASSIVA)` — CFrames EXATAS Rufus14
- Parâmetros: `dhposepeed=25`, `dhanimspeed=20`, `dhtime=0.65`
- 15% chance de empurrão ao passar perto de NPC/jogador
- VFX: esfera Neon com TweenService (base: His Cube AbbilityClient)

### Posições dos botões (scale-only, sem pixel)
```lua
-- His Cube ContextActionService (base)
SetPosition("PassivePunch", UDim2.new(0.5,  0, 0,    0))
SetPosition("PassiveBlock", UDim2.new(0.35, 0, 0,    0))
SetPosition("PassiveDash",  UDim2.new(0.5,  0, -0.5, 0))
```

## 🎮 SISTEMA DE SELEÇÃO DE PERSONAGENS

### CharacterSelectionServer V2
```
PlayerAdded → cria Folder "UserId's OwnedCharacters"
           → carrega DataStore por personagem
           → FireClient "CheckOwnedCharacters"

OnServerEvent(player, chosenCharacter)
   → verifica cooldown (LockedPlayers)
   → verifica preço e posse
   → MorphCharacter()
       → clona modelo de ReplicatedStorage/Characters
       → insere BoolValue "CharacterSelected"
       → FireClient "CharacterLocked"
       → task.wait(0.1) → GiveTools()
       → humanoid.Died → LockedPlayers[userId]=nil
                       → FireClient "CharacterUnlocked"
```

### GiveTools()
```
Lê ReplicatedStorage/CharacterTools/<nome>/
Para cada filho:
  - Se Tool → clona direto para Backpack
  - Se StringValue → busca em ServerStorage/Tools pelo nome
```

### CharacterPrices (ModuleScript)
```lua
{ Noob=0, Calebe=0, Danilo=0, ICAN_REA0=0, ["Gui a Bomba"]=0 }
```

### Mensagens do RemoteEvent
| Mensagem | Direção | Significado |
|---|---|---|
| `"CheckOwnedCharacters"` | Server→Client | Atualizar UI de posse |
| `"CharacterLocked"` | Server→Client | Personagem escolhido, ativa passivas |
| `"CharacterUnlocked"` | Server→Client | Morreu, pode escolher novamente |
| `"NotOwned"` | Server→Client | Sem cash suficiente |
| `<nome do personagem>` | Client→Server | Jogador selecionou personagem |

## 🛠 REGRAS DE TOOLS

### Propriedades obrigatórias
```lua
tool.CanBeDropped  = false  -- SEMPRE
tool.RequiresHandle = true  -- SEMPRE
tool.ToolTip       = "Nome — Descrição"
-- Handle com nome exato "Handle" (case-sensitive)
```

### Proibições absolutas
- ❌ `part:Destroy()` → usar `Transparency=1` ou `Parent=nil`
- ❌ `math.random()` / randomize → valores pré-definidos
- ❌ Destruir Humanoides → `humanoid.Health=0` apenas controlado
- ❌ GUIs completas → apenas botões simples para habilidades Extra

### Debounce padrão
```lua
tool.Activated:Connect(function()
    if not tool.Enabled then return end
    tool.Enabled = false
    -- habilidade
    task.wait(cooldown)
    tool.Enabled = true
end)
tool.Unequipped:Connect(function()
    tool.Enabled = true
end)
```

### Versionamento
`V1 → V2 → V3` sequencial. Sempre informar qual script foi substituído.
